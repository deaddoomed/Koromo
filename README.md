# Koromo Project

Koromo is the codename for a cozy game project with relationship simulation and management simulation mechanics inspired on games like Stardew Valley, Persona and Animal Crossing.
The players is in charge of a food bussiness, choosing between many food options and able to customize its asthetic.
The establishment management implies buying supplies, find people to help in the restaurant and leading with bills and management.

> [!IMPORTANT]
> Cloning instructions:
> 1. Create an 3D URP project for Unity 6.3
> 2. Clone this repository
> 3. Copy the projects files into the repository folder without overwrite existing files.
> 4. Delete previous project from Unity Hun
> 5. Add repository folder as project in Unity Hub

## Mechanics implemented

- Basic UI with Stats and Clock
- Time and Calendar System
- Camera controller

> [!NOTE]
> Newtonsoft is needed. To install it, go to:
> Window > Package Management > Package Manager > + > Installa package by name /n
> Use: com.unity.nuget.newtonsoft-json
